  • p - do zwykłego tekstu
  • label - do opisu każdego input-u (o input chodzi mi input użytkownika, a nie tag input)
  • br - do przerzucania rzeczy do następnej linijki bądź do tworzenia większej przestrzeni pomiędzy input-ami
  • input - text
  • textarea
  • input - email
  • input - password
  • input - date
  • input - radio
  • input - range
  • select
  • checkbox
  • button bądź input - button => typy: reset oraz submit
